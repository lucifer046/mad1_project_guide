## HOW TO RUN THE APP

```bash
# 1. Go to project folder
cd trekking_app

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the app
python app.py

# 4. Open browser
# Go to: http://127.0.0.1:5000

# Default admin credentials:
# Email: admin@trek.com
# Password: admin123
```
